Zero-js

The smallest 'framework' you'll ever use

View on GitHub Download .zip Download .tar.gz

The future is now.

zero.js (V1.7.9) is the world's smallest javascript framework, compressed to only 0kb. It requires no transpiling, no bable runtimes, and uses the javascript syntax you already know!

Simply write your functions in plain old javascript (POJS) and watch as they run seamlessly in the browser!

Just run npm install zero-js and get started changing the world.

This framework was carefully crafted by smitec (smitec.io) while waiting for an npm install to finish running.
